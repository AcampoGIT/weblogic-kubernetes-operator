Oracle WebLogic Server Kubernetes Operator

The Oracle WebLogic Server Kubernetes Operator provides a mechanism to manage the operations of WebLogic Domains in a Kubernetes Cluster. This includes creation of Domains, automation of Domain startup, scaling of WebLogic Clusters, management of Ingress rules, and integration with other common tools used in the Kubernetes ecosystem, like ELK for example.

This release is a Technology Preview, or "alpha" quality release.
